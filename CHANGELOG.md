# Changelog

All notable changes to LastSnap Free will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-16

### Added
- Initial release of LastSnap Free version
- MP4 video upload with drag-and-drop support
- Last frame extraction using OpenCV
- Real-time upload progress indicator
- Responsive React frontend with modern UI
- Flask backend API
- Docker and Docker Compose support
- Social media sharing functionality
- One-click download of extracted frames
- Health check endpoint for backend
- Comprehensive README documentation
- MIT License for open-source use
- Quick start script for easy setup

### Features
- ✅ No authentication required
- ✅ No rate limiting
- ✅ Free and open-source
- ✅ Docker-ready deployment
- ✅ Mobile-responsive design
- ✅ CORS enabled for all origins

### Technical Stack
- **Frontend**: React 19, Vite, Modern CSS
- **Backend**: Python 3.11, Flask, OpenCV
- **Deployment**: Docker, Docker Compose
- **License**: MIT
